# 🔍 ScanLogs — Windows Log File Scanner

Лек и бърз PowerShell инструмент за сканиране на `.log` файлове на Windows системи. Показва брой и размер на системни и потребителски лог файлове директно в терминала.

---

## 📋 Описание

**ScanLogs** обхожда целия `C:\` диск и категоризира всички `.log` файлове:

- **Системни логове** — файлове в `C:\Windows\`
- **Лог файлове на приложения** — всичко извън `C:\Windows\` (Program Files, Users и др.)

Версията **x86** допълнително идентифицира логове от `SysWOW64` (32-битови компоненти) и автоматично се рестартира в 32-битов PowerShell, ако е нужно.

---

## 📁 Файлове

| Файл | Описание |
|------|----------|
| `ScanLogs.ps1` | Стандартна (64-bit) версия |
| `ScanLogs_x86.ps1` | 32-bit версия с поддръжка на SysWOW64 |

---

## ⚙️ Изисквания

- Windows 7 / 8 / 10 / 11
- PowerShell 3.0 или по-нова версия
- Препоръчително: **Стартиране като Администратор** (за достъп до защитени папки)

---

## 🚀 Как да използвате

### Метод 1 — PowerShell директно

```powershell
# Стандартна версия
.\ScanLogs.ps1

# 32-bit версия
.\ScanLogs_x86.ps1
```

### Метод 2 — Десен клик

1. Кликни с десен бутон върху `.ps1` файла
2. Избери **"Run with PowerShell"**

### Метод 3 — Ако скриптовете са блокирани

```powershell
powershell -ExecutionPolicy Bypass -File .\ScanLogs.ps1
```
